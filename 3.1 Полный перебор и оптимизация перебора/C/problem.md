# Сочетания с повторениями

Выведите число сочетаний с повторением С‾(n,k).

## Формат ввода

В первой строке находятся два числа n (1≤n≤4), k (1≤k≤4).

## Формат вывода

Выведите ответ на задачу.

### Пример 1

Ввод

    1 1
    

Вывод

    1
    

### Пример 2

Ввод

    4 3
    

Вывод

    20
    

### Пример 3

Ввод

    2 2
    

Вывод

    3
