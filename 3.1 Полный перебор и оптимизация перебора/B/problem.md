# Сочетания

Выведите число сочетаний C(n,k).

## Формат ввода

В первой строке находится два числа n (1≤n≤7), k (1≤k≤7).

## Формат вывода

Выведите ответ на задачу.

### Пример 1

Ввод

    3 2
    

Вывод

    3
    

### Пример 2

Ввод

    7 5
    

Вывод

    21
    

### Пример 3

Ввод

    1 1
    

Вывод

    1
